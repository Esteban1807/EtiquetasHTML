En este blog vamos a evidenciar algunas de las etiquetas más importantes que tenemos en HTML, organizadas por categorías para facilitar su comprensión.

Etiquetas Estructurales

  • <html>

    • Definición: Define el inicio y final de un documento HTML.

    <html>Contenido</html>
  • <header>

    • Definición: Define la cabecera de una página o sección.

    <header>Contenido</header>
  • <body>

    • Definición: Contiene todo el contenido visible.

    <body>Contenido visible</body>
  • <main>

    • Definición: Contenido principal del documento.

    <main>Contenido principal</main>
  • <footer>

    • Definición: Pie de página.

    <footer>Pie</footer>

Etiquetas de Texto

  • <p>

    • Definición: Define un párrafo.

    <p>Este es un párrafo</p>

    Ejemplo real:

    Este es un párrafo en funcionamiento.

  • <strong>

    • Definición: Indica texto importante.

    <strong>Importante</strong>

    Texto importante

  • <em>

    • Definición: Texto con énfasis.

    <em>Énfasis</em>

    Texto con énfasis
