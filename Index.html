Etiquetas Estructurales

  • <html>

    • Definición: Define el inicio y final de un documento HTML.

    Sintaxis:

    <html>Contenido</html>

    Uso adecuado: Se utiliza como el contenedor principal de la página.

    Buenas prácticas: No debe faltar, ya que es esencial para estructurar el contenido de una página web.

  • <header>

    • Definición: Define la cabecera de una página o sección.

    Sintaxis:

    <header>Contenido</header>

    Uso adecuado: Se usa para títulos o menús.

    Buenas prácticas: No usarlo solo para decorar.

  • <body>

    • Definición: Define el contenido visible de la página web.

    Sintaxis:

    <body>Contenido visible de la pagina web</body>

    Uso adecuado: Se usa para contener todo el contenido que se muestra al usuario, como textos, imágenes, formularios, etc.

    Buenas prácticas: No debe contener elementos fuera del contenido principal de la página.

Etiquetas de Texto

  • <p>

    • Definición: Define un párrafo de texto.

    Sintaxis:

    <p>Texto del parrafo</p>

    Uso adecuado: Para contener y mostrar bloques de texto en una página web.

    Buenas prácticas: Evitar usarlo para otros elementos no relacionados con texto.

  • <p>

    • Definición: Define un párrafo.

    Ejemplo:

    <p>Este es un párrafo</p>

    Resultado:

    Este es un párrafo
