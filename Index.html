Etiquetas Estructurales

  • <header>

    • Definición: Define la cabecera de una página o sección.

    Sintaxis:

    <header>Contenido</header>

    Uso adecuado: Se usa para títulos o menús.

    Buenas prácticas: No usarlo solo para decorar.

Etiquetas de Texto

  • <p>

    • Definición: Define un párrafo.

    Ejemplo:

    <p>Este es un párrafo</p>

    Resultado:

    Este es un párrafo

Listas

<ul>

Definición: Lista desordenada.

   <li>Elemento 2</li>
</ul>
